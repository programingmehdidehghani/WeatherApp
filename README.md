# weatherApp project
## by this App you can see what's the weather like in your Location
### The following technologies are used in this project:
+ Compose
   + for show list weather I use LazyRow in JetpackCompose
+ CleanArchitecture
+ Rest_Api(Restrofit)
+ Room
+ Dagger-Hilt(Di)
+ Mvvm (in presentationLayer)
