# weatherApp project

### The following technologies are used in this project:
+ Compose
   + for show list weather I use LazyRow in JetpackCompose
+ CleanArchitecture
+ Rest_Api(Restrofit)
+ Room
+ Dagger_Hilt(Di)
+ Mvvm (in presentationLayer)
